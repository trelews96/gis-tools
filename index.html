Chrome Extensions

These are lightweight Chrome extensions that run automatically in the background on ArcGIS Map Viewer pages. Unlike the bookmarklet tools above, extensions require a one-time install but need no action each session — they are always on.

🛡️

ArcGIS Editor: Contingent Values Neutralizer

Eliminates Map Viewer editor lag caused by large contingent-value matrices. Speeds up both edit form initialization and in-editor field interaction. Server-side validation still runs on save.

✅ Active on Map Viewer only
Install
📦 Download arcgis-cv-killer.zip
Setup Steps
1

Unzip the downloaded file

Right-click arcgis-cv-killer.zip → Extract All. Place the resulting folder somewhere permanent — Chrome loads the extension directly from this folder, so don't delete or move it after install.

2

Open Chrome's extension manager

In Chrome, navigate to chrome://extensions in the address bar, or go to the three-dot menu → Extensions → Manage Extensions.

3

Enable Developer Mode

Toggle the Developer mode switch in the top-right corner of the extensions page. This unlocks the ability to load extensions from your own files.

4

Load the extension

Click Load unpacked (top-left), then navigate to and select the arcgis-cv-killer folder you unzipped in Step 1. The extension appears in the list immediately.

5

Done — no further action needed

The extension runs automatically every time you open a Map Viewer page. You do not need to click anything to activate it each session.

✅ Verify it's working
1. Open a Map Viewer page and press F12 to open DevTools → Console tab.
2. You should see 🛡️[CV-Killer] Active @ ... near the top of the console output.
3. Click edit on a feature in a contingent-value layer. The editor should open noticeably faster than before.
⚠️ The extension folder must stay in place after install. If you move or delete it, the extension will show an error in chrome://extensions and stop working until you reload it from its new location.
⚠️
Requirement: GIS Tools only works on an ArcGIS web map page (Map Viewer or Experience Builder). Clicking the bookmark on any other page will have no effect. Make sure your web map is fully loaded before launching.

Launching the Tool Panel

1

Open your ArcGIS web map

Navigate to your Map Viewer or Experience Builder app and wait for the map to fully load.

2

Click the 🔧 GIS Tools bookmark

Click the bookmark in your bookmarks bar. The dark GIS Tools panel will appear in the top-left corner of the page.

3

Click any tool to load it

Tools load on demand from GitHub when you click them. The first load may take a moment — watch the status bar at the bottom of the launcher for confirmation before using the tool.

4

Close tools when finished

Use the ✕ on individual tool panels to close them, or click ⬛ Close All Tools at the bottom of the launcher to reset everything at once.

Panel Controls

✋

Drag to Reposition

Click and drag the panel header bar to move the launcher anywhere on screen so it doesn't block your map.

▾

Collapse / Expand

Click the ▾ button in the header to collapse the launcher to just the title bar, freeing up screen space while keeping it accessible.

✕

Close the Launcher

The red ✕ button closes the launcher and all open tools. Click the bookmark again at any time to relaunch it.

📂

Tool Groups

Click any group header (Geometry, Packages, etc.) to collapse or expand that category to keep the panel tidy.

General Tips & Gotchas

⚠️ Tools modify live map data. Always confirm you are on the correct web map and work order before making edits.
💡 If a tool doesn't open or behaves unexpectedly, try refreshing the page and clicking the bookmark again — this fully reinitializes the host.
💡 The launcher resets on page refresh. This is by design — each session starts clean.
⚠️ On slow connections, tools may take a few seconds to load the first time. Wait for the ✅ confirmation in the status bar before interacting with the tool panel.
✏️ Geometry & Feature Editing

GIS Edit Tools
Move features, edit vertices, split lines, and copy features on the map
▾
Description
A lightweight editing toolbar that attaches to your ArcGIS web map without replacing the native editor. Provides four focused editing modes in a single draggable panel: moving features and vertices with automatic snapping, locking edits to a single feature when features overlap, splitting lines at a point location, and placing copies of existing features at clicked locations.
Move & Vertex Tools
  1. 1Click Enable Tool to activate editing mode.
  2. 2Choose Point Mode to move point features, or Line Mode to move individual line vertices.
  3. 3Click the feature or vertex to select it, then click the destination to move it.
  4. 4In Line Mode, use Add Vertex / Delete Vertex to insert or remove vertices on any visible line.
  5. 5Toggle Show Vertices to display all line vertices as a colored overlay.
Tips & Gotchas
⚠️ Use Single Feature Lock whenever features overlap — without it, coincident vertices on multiple lines may all move together.
💡 Snapping is automatic — destinations will snap to the nearest point feature or line vertex within the configured tolerance.

Path Editor
Guided sequential attribute editing across multiple map features
▾
Description
Rather than clicking features one by one on the map, Path Editor lets you define a selection, configure which layers and fields to edit, and step through each feature in a structured queue. Supports single click, drawn path, and polygon selection modes.
Tips & Gotchas
💡 Save your layer/field configuration after setup — repeat workflows can be reloaded instantly with the Load button.
⚠️ Bulk Edit applies the same values to ALL selected features at once — verify your filter is correct before using it.
